**VPOS Touch**

The following components are part of the Touch device setup. Each plays a specific role during installation and operation:

| Name                              | Description                                                                                    | Image                                                                                                   |
| :-------------------------------- | :--------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------ |
| **Touch Device**                  | The Nayax VPOS Touch Marshall device                                                           | <Frame>![](/images/docs/sales-kit/images1.png)</Frame>  |
| **Device Connector**              | Used to connect power, cables, and additional accessories required for the device's operation. | <Frame>![](/images/docs/sales-kit/images2.png)</Frame> |
| **LAN (Ethernet Cable Port)**     | Enables a wired connection to a router for internet access.                                    | <Frame>![](/images/docs/sales-kit/images3.png)</Frame> |
| **Antenna**                       | Allows the attachment of the cellular antenna for network communication.                       | <Frame>![](/images/docs/sales-kit/images4.png)</Frame> |
| **SIM**                           | SIM card for enabling data connectivity in Nayax Touch VPOS devices.                           | <Frame>![](/images/docs/sales-kit/images5.png)</Frame> |
| **Chipi-X USB2Com adaptor cable** | USB-to-serial adaptor cable for connecting Nayax Touch VPOS devices.                           | <Frame>![](/images/docs/sales-kit/images6.png)</Frame> |
| **24V DC power adaptor**          | 24V DC power adaptor for powering Nayax Touch VPOS units.                                      | <Frame>![](/images/docs/sales-kit/images7.png)</Frame> |
