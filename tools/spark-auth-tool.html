Encrypt or decrypt the cipher used in the StartAuthentication method. Choose between the current v2 flow (with timestamp) and the legacy v1 flow.

New authentication (v2) — Cipher plaintext: TxId(36) = Random(17) + Timestamp(YYMMDDhhmm). Encrypted with AES-256-ECB using the last 32 chars of your token. Header: TransactionSignature: SHA256(TxId;SignKey).
1Select Encrypt to generate a new cipher, or Decrypt to inspect an existing one.
2Enter your token, token ID and sign key. Hover ? icons for field details.
3Fill in the transaction details. Use Generate to create the GUID and random string.
4Click Generate cipher + headers and copy the results.

StartSession

Build or inspect the cipher for the StartSession callback. Nayax calls your endpoint — you respond with an encrypted cipher.

Flow: Nayax POSTs to your endpoint with HwSerial, MachineId, TokenId and a 27-char Random. You respond with a cipher: TxId(36)=Random(27) → AES-256-ECB. No timestamp in StartSession cipher.
1Nayax calls your endpoint. Enter the values they send (HwSerial, MachineId, TokenId, Random) in section 1.
2In section 2, enter your secret token and generate a 36-digit transaction ID.
3Click Generate StartSession cipher. Return the response body JSON to Nayax.

Header Signature

Calculate the HTTP authentication headers for StartAuthentication requests. Choose between the current v2 TransactionSignature and the legacy v1 Signature flow.

Computes TransactionSignature for New Auth (v2): SHA256(SparkTransactionId;SignKey)
1Enter the Spark transaction ID (same GUID used in the request body) and your sign key.
2Click Calculate headers to get the IntegratorId and TransactionSignature values.
The same UUID v4 used in the request body. Signature = SHA256(this;SignKey).
16-char secret key from Nayax. Combined with TxId to produce the TransactionSignature header.
Your numeric integrator ID. Sent as the IntegratorId HTTP header.

Returns both HTTP headers ready to attach to your request.

Computes Signature for Legacy Auth (v1): SHA256(minifiedJSON;HeaderHashKey)
1Paste the full request body JSON — it's minified automatically before hashing.
2Enter the header hash key and integrator ID, then click Calculate legacy signature.
Full request body you're sending. Auto-minified via JSON.stringify() before hashing. Must exactly match the HTTP body.
16-char key from Nayax for legacy auth. Used to compute: SHA256(minifiedBody;HeaderHashKey).
Your numeric integrator ID sent as IntegratorId HTTP header.

Returns the minified JSON and Signature header value.
