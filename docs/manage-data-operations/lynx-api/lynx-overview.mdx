Lynx is Nayax's Management API. It allows you to integrate with Nayax Core and manage your machines, devices, and inventory from your own environment. This section of the documentation offers guides and resources to help you interact with the Lynx API for different tasks, such as inventory management, machine configuration, and more.

## Environments

All guides in this section use QA URLs in their examples. Replace the hostname when going to production.

| Environment | Base URL |
|-------------|----------|
| QA | `https://qa-lynx.nayax.com/operational` |
| Production | `https://lynx.nayax.com/operational` |

## Key Benefits of the Lynx API

The Lynx API offers a wide range of benefits for businesses looking to automate and streamline their operations:

* **Simple Integration**: Easily integrate the Lynx API with your existing in-house systems.
* **Comprehensive Capabilities**: Perform a variety of actions, including:
  * Inventory management and optimization
  * Machine and device configuration
  * Executing remote actions and commands
  * Managing prepaid cards

## Lynx Use Cases

The Lynx API supports several real-world use cases, for example:

* **ERP System Integration**: Integrate with ERP systems for real-time updates and management.
* **Prepaid Card Top-Up**: Add credit to prepaid cards using automated workflows.
* **Consumer Management**: Automate prepaid card management for workplace or retail environments.
* **Inventory Management**: Create pick lists, perform stock updates, and optimize inventory levels across multiple machines.
