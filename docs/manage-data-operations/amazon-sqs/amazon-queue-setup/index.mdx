To set up Amazon SQS integration with Nayax, we recommend exploring the following documentation pages:

1. [**Create New Queue**](/docs/create-new-queue): Learn how to set up a queue to receive data securely and efficiently.
2. [**Set Up Queue Permissions**](/docs/set-up-queue-permissions): This guide covers configuring access controls to manage queue security.
3. [**Create User for Queue Acess**](/docs/create-user-for-queue-access): Follow this guide to create and configure an IAM user with the appropriate permissions.

## Prerequisites

Before proceeding, ensure the following:

* You have an [**Amazon AWS account**](https://aws.amazon.com/free/?all-free-tier.sort-by=item.additionalFields.SortRank\&all-free-tier.sort-order=asc).
* You are logged in to the **AWS Management Console**.

Once these prerequisites are met, follow the steps below to set up the Amazon SQS Queue.
