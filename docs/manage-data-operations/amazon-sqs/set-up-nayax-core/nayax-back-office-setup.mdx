Nayax Core supports delivery of transaction and operational data through multiple channels. You can configure real-time delivery to Amazon SQS, or set up periodic batch delivery to an SFTP server or email address.

To set up Nayax Core, we recommend reviewing the following documentation pages:

* [**Set Up Transaction Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-transaction-delivery): Configure real-time delivery of transaction data to Amazon SQS, or set up periodic delivery via SFTP or Email.
* [**Set Up EVA DTS DEX/DDCMP Audit File Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-eva-dts-dexddcmp-audit-file-delivery): Enable audit file delivery in compliance with EVA DTS standards.
* [**Set Up Events Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-events-delivery): Configure delivery of system events for real-time tracking and monitoring.
* [**Set Up Encryption on Amazon SQS Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-encryption-on-amazon-sqs-delivery): Implement encryption for secure data transmission.

By following these guides, you can fully configure Nayax Core for all supported delivery methods.
