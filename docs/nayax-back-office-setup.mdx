---
title: "Set Up Nayax Core"
---

Nayax Core facilitates seamless data delivery to Amazon SQS. By configuring the settings in Nayax, you can ensure efficient and secure delivery of critical data, including transactions, EVA DTS DEX/DDCMP audit files, and events, while enabling encryption for enhanced data security.

To set up Nayax Core integration with Amazon SQS, we recommend reviewing the following documentation pages:

1. [**Set Up Transaction Delivery**](/docs/set-up-transaction-delivery): Learn how to configure the delivery of transaction data to Amazon SQS.
2. [**Set Up EVA DTS DEX/DDCMP Audit File Delivery**](/docs/set-up-eva-dts-dexddcmp-audit-file-delivery): Follow this guide to enable audit file delivery in compliance with EVA DTS standards.
3. [**Set Up Events Delivery**](/docs/set-up-events-delivery): Configure the delivery of system events for real-time tracking and monitoring.
4. [**Set Up Encryption on Amazon SQS Delivery**](/docs/set-up-encryption-on-amazon-sqs-delivery): Understand how to implement encryption for secure data transmission.

By following these guides, you can fully optimize your Nayax Core for Amazon SQS integration.
