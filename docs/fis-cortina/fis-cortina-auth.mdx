---
title: "Authentication"
metadata:
  og:description: "Before performing any fiscal transaction, Nayax must establish a secure session with the Fiscal Integrator using an authentication token."
---

Nayax establishes a secure session with the Fiscal Integrator before any fiscal transaction (Register or Void). It uses an authentication token to initiate this session.

Use this token during the Start Session request to authorize subsequent fiscal operations.

<Note>
**Authentication Token**

The Nayax support team will provide the authentication token.
</Note>
