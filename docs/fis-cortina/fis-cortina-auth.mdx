Before performing any fiscal transaction (Register or Void), Nayax must establish a secure session with the Fiscal Integrator.

This token is used during the Start Session request to authorize subsequent fiscal operations.

<Note>
**Authentication Token**

The Nayax support team will provide the authentication token.
</Note>
