Nayax logs Marshall C SDK changes on this page as they ship. Subscribe to
the RSS feed using the icon at the top of the page to get notified of new
entries.

Release notes are organized into two categories to help you evaluate the
impact of each update:

* **Improvements**: New APIs, expanded feature support, and performance
  enhancements.
* **Fixes**: Resolved bugs and reliability improvements.

<Accordion title="Improvements" icon="rocket">
### Transfer data and technician commands

Site ID and brand name are now transferred to the machine via transfer data, and a new technician-mode command opens the door remotely.

***

### Error handling

Enhanced error handling with improved validation and more descriptive messages.
</Accordion>

<Accordion title="Fixes" icon="wrench">
### High com_id issue

Fixed a bug related to a high `com_id` value.
</Accordion>

</Update>
