The `doMiniSettlement` method is used to settle transactions.

<Note>
  Supported processors: CG, PC.
</Note>

## Request

<Note>
**JSON-RPC 2.0**

All methods in TweezerComm follow a JSON-RPC 2.0 structure. See [Integration ](/docs/integration-tweezer) for more details on how the requests are structured and sent.
</Note>

In the request, ensure to configure the following:

* The `method`field must be set to `doMiniSettlement` .
* The service must be set to `ashrait` in the `params`.

혻 혻 "method":"doMiniSettlement",
혻 혻 "params":[
혻 혻 혻 혻 "ashrait"
