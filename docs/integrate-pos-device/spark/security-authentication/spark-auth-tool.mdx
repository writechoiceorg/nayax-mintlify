Use this tool to handle Spark authentication without writing any code. Everything runs in your browser, so your token and sign key never leave your machine.

Use the tabs at the top to switch between **StartAuthentication**, **StartSession**, and **Header Signature**. On the StartAuthentication tab, the **Auth** and **Legacy Auth (v1)** sub-tabs cover the current and legacy signature flows.
