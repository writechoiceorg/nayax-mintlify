The EMV Core SDK provides API methods to handle payment processing and device management. The SDK provides the Kiosk or 'Point Of Sale' (POS) system developers a pre-certified EMV Payment solution, allowing them to develop their ‘Point Of Sale’ or Payment Kiosk easily and quickly, while concentrating in the user interface and business aspects, and without dealing with the complexity of the EMV transactions processing., and ensure seamless integration with Nayax’s backend services.

This section guides you through using the SDK features. See the following pages for more information:
