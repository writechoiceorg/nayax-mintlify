Cortina SQR 使用 `Settlement` 命令通知集成方设备端结算流程的结果(意味着产品已经被出货),以启动与 Cortina 支付提供商的结算流程。

此命令对于进行交易确认或拒绝处理至关重要。

## 要求

Nayax 的服务器会向您配置的 `Settlement` 端点发送 POST 请求,格式如下:

`(customer's URL)/Cortina/StaticQR/Settlement`

## 回复

客户的支付方式发送给 Nayax 服务器的 Settlement 响应。有效负载应如下所示:

```json
{
  "Status": {
    "Verdict": "Declined",
    "Code": 1,
    "StatusMessage": "Insufficient balance"
  }
}
```

### 响应参数

下表描述了 API 响应中返回的字段：

<table>
  <thead>
    <tr>
      字段
      类型
      描述
    </tr>
  </thead>

  <tbody>
    <tr>
      交易状态及错误代码
    </tr>

    <tr>
      `Verdict`
      String

      <td style={{ textAlign: "left" }}>
        表示**请求最终状态的**常量字符串：

        <ul>
          <li>**`Approved`**</li>
          <li>**`Declined`**</li>
        </ul>
      </td>
    </tr>

    <tr>
      `Code`
      Int(32)

      <td style={{ textAlign: "left" }}>
        拒绝代码（Decline Code）。条件性字段——仅当 `Verdict` 为 “Declined” 时才会出现。有效代码包括：

        <ul>
          1：余额不足
          5：疑似欺诈
          6：系统一般性故障
          7：金额无效
          8 / 998：格式/解析错误或验证错误
          992：超时异常
          997：缺少必填参数
          999：一般性异常
          有关其他状态和内部错误（例如 40、990、1011），请参阅完整列表。
        </ul>
      </td>
    </tr>

    <tr>
      `StatusMessage`
      String(255)

      <td style={{ textAlign: "left" }}>
        包含**交易状态消息**或有关结果的其他不同数据的自由文本字段。
      </td>
    </tr>

    <tr>
      `CustomDeclineCode`
      String

      <td style={{ textAlign: "left" }}>
        用于指定自定义拒绝代码（例如与 Soft Decline 或 Switch Interface 相关的代码）。可能反映收单机构（acquirer）的响应代码。
      </td>
    </tr>

    <tr>
      忠诚度和信用信息
    </tr>

    <tr>
      `RegularCreditType`
      Int(32)

      <td style={{ textAlign: "left" }}>
        定义 `RegularCredit` 的计量单位（枚举值）：

        <ul>
          0：信用额度以交易次数计（count）。
          1：信用额度以金额计（currency）。
        </ul>
      </td>
    </tr>

    <tr>
      `RegularCredit`
      Double

      <td style={{ textAlign: "left" }}>
        根据 `RegularCreditType` 计算的常规信用额度值（金额或交易次数）。使用小数位数 2。
      </td>
    </tr>

    <tr>
      `RevalueCredit`
      Int(32)

      <td style={{ textAlign: "left" }}>
