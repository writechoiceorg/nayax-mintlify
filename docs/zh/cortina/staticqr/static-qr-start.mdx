## 要求

Cortina 支付方式将向 Nayax 服务器发送 `Start` 命令,以通知其交易已开始。该调用的格式如下:

`https://lynx.nayax.com/payment/v2/transactions/cortina/(payment method's name)/start`

有效载荷包含以下详细信息：

<CodeGroup>
  ```json JSON-TerminalID
  {
    "AppUserID": "19",
  	"TerminalId": "0456789456789456",
    "TransactionId": "123456789qwertyuioasdfghjkl",
    "SecretToken": "mrV3U3nsgGFrE3w5-wnBo_WCLPce-pZ1awRvTVTkungMIKThTVbj_fiXdfoGclhn0"
  }
  ```

  ```json JSON-UniQR
  {
    "AppUserID": "19",
  	"UniQR": "https://qr.nayax.com/v1/rWg_p6pZh1pwjvbkGrmBLy3KgwSz3dq-y63r5FPxk2NOWw2",
    "TransactionId": "123456789qwertyuioasdfghjkl",
    "SecretToken": "mrV3U3nsgGFrE3w5-wnBo_WCLPce-pZ1awRvTVTkungMIKThTVbj_fiXdfoGclhn0"
  }
  ```


### Remote Vend:产品选择

The following applies to the **PreSelection flow** only. To specify which product to vend, include a `"code"` field inside the `"Products"` array with the corresponding product index.

脉冲机器代码编号

对于 Pulse 阵列机器(除 Pulse Line 之外的任何 Pulse 配置),`"code"` 编号从 0 开始。例如,若为:

<Frame>
  图像
</Frame>

- `"Code": 0` → "Ultimate" 产品
- `"Code": 1` → "Superior" 产品
- `"Code": 2` → "Regular" 产品

如果您发送的数字超出虚拟机数组的范围,我们的服务器将选择列表中的第一项(`Code: 0`)。

对于 Pulse 1–6 配置(Pulse Line),请使用 `"PulseLineNumber"` 而非 `"code"`,且计数从 1 开始(而非 0)。

## 回复

Nayax 服务器将发送给支付方式的 `Start` 响应。其负载应如下所示:

```json
{
  "Status": {
    "Verdict": "Approved",
    "StatusMessage": "Cortina V2 Stub Tester"
  }
}
```
