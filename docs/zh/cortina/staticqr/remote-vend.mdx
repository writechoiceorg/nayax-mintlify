---
title: 产品远程销售
---

Remote Vend 允许集成方使用 `Start` 调用,通过所选产品发起一笔交易。

对于 MDB 或 Marshal 机器，所选产品可以作为产品代码发送；对于 Pulse 机器，则可以作为特定的脉冲线 ID 发送。

<Warning>
  此功能仅支持以下协议:

  - MDB level 3(需要物理机器支持 MDB level 3,并且在 Nayax Core 上有匹配的配置)
  - Pulse

  不支持以下情况:

  - MDB level 1 / 2
  - 马歇尔
</Warning>

<Note>
  此功能目前仅在 `VPOS\ONYX` 设备上受支持。
</Note>

## 如何实施

发送 Start 请求时，添加一个名为“Products”的属性，该属性包含一个或多个产品，如下所示：

### 脉冲机

#### 它支持从哪个固件版本开始？

4.0.10.23

#### 通话形式是什么？

Cortina V1/V2：

```json
{

"AppUserId":"260760",

"TerminalId":"0456789456789456",

"TransactionId":"12345129",

"SecretToken":"1234OqgODDqYgkJbSU-Vxw0MkZM1XjHvbQeSTnHmhwltUM_5kB-ZSgofkCxNk3ZN0",

"Products":

[
 {
      
 "Code":2

 }

 ]

}
```

### 非Pulse（MDB、Marshall等）

- MDB - 请注意，远程销售功能仅在 MDB 3 级机器上受支持（协议限制）。
