Nayax 提供三种灵活的集成方法，以满足各种运营需求，并实现与生态系统的无缝集成：

<Columns cols={3}>
  <Card icon="link" title="山猫" href="/docs/manage-data-operations/lynx-api/lynx-overview">
    连接到 Nayax Core 管理平台并从您自己的环境管理您的机器、设备、库存和预付卡。
  </Card>

  <Card icon="link" title="马歇尔" href="/docs/integrate-pos-device/marshall/marshall-sdk">
    利用物理连接方式，将任何无人值守机器与 Nayax 的支付设备（VPOS Touch、Onyx）连接。
  </Card>

  <Card icon="link" title="科尔蒂纳" href="/docs/cortina/overview-cortina">
    如果您的组织有自己的客户应用程序、会员卡或闭环支付方式，您可以通过快速的 Cortina 集成在任何 Nayax POS 上轻松接受它。
  </Card>

  <Card icon="link" title="Amazon SQS 设置" href="/docs/manage-data-operations/amazon-sqs/overview">
    通过 Amazon SQS 服务检索有关您的销售和运营的实时信息，包括交易、机器遥测和警报。
  </Card>

  <Card icon="link" title="EMV核心" href="/docs/integrate-pos-device/emv-core/overview">
    将任何无人值守机器与嵌入式 Nayax 支付设备（Uno 线）连接起来。
  </Card>

  <Card icon="link" title="TweezerComm" href="/docs/integrate-pos-device/tweezercomm/overview-tweezer">
    利用物理连接方式，将 Nayax 的有人值守支付设备（Nova 系列）与您的销售点（POS）系统集成。
  </Card>

  <Card icon="link" title="火花" href="/docs/integrate-pos-device/spark/spark">
    远程将您的平台与 Nayax 设备集成（服务器到服务器集成）。非常适合 POS 和平台集成。
  </Card>
</Columns>
