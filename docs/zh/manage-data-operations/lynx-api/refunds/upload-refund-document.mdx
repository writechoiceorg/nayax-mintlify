如果客户直接联系账单提供商申请退款，则在接受退款时必须将 `IsRefundedExternally` 设置为 `true`，并将账单提供商收到的文件添加到 `RefundedDocumentUrl` 中。

在 Nayax Core 中，选择**“外部退款”**接受或拒绝退款时，即可设置此项。您可以上传退款文件。参见下图：

## Lynx API

使用 Lynx API，您还可以上传退款文档以获取 `RefundedDocumentUrl`。以下端点用于此目的：

<EndpointCard title="上传退款文件" href="/reference/lynx/payment/upload-refund-documentation" method="post" />

<Warning>
  验证

  请参阅本文档的[“安全和令牌”](/docs/manage-data-operations/lynx-api/security)页面，了解如何访问您的令牌以及如何正确使用它们来验证您的 API 请求。
</Warning>

使用[“上传退款文档”](/reference/lynx/payment/upload-refund-documentation)端点上传账单提供商收到的文件。请参阅以下请求示例：

```sh Request
curl --request POST \
     --url https://qa-lynx.nayax.com/operational/v1/payment/upload-refund \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json' \
     --header 'content-type: application/json' \
     --data '
{
  "FileName": "string",
  "FileData": "string",
  "TransactionId": 0,
  "SiteId": 0,
  "MachineAuTime": "2024-10-10T16:30:37.179Z"
}
'
```

下表详细说明了请求的主体参数。

| 姓名       | 类型     | 描述                                                                                         |
| -------- | ------ | ------------------------------------------------------------------------------------------ |
| 文件名      | string | 正在上传的文件的名称。                                                                                |
| 文件数据     | base64 | 正在上传的文件的 base64 编码内容。您可以使用[PDF 到 base 64 在线转换器](https://base64.guru/converter/encode/pdf)。 |
| 交易ID     | int64  | 与文件关联的交易的唯一标识符。                                                                            |
| 站点编号     | int32  | 发生交易的站点的唯一标识符。                                                                             |
| 机器AuTime | 日期时间   | 机器的授权时间与交易相关联。                                                                             |

<Note>
  `TransactionId`、`SiteId` 和 `MachineAuTime` 必须与退款请求中发送的值相同。
</Note>

一旦请求成功完成，响应将包含上传文档的 URL。

```json Response
{
  "FileURL": "string"
}
```

Now you can add this URL to the `RefundedDocumentUrl` flag when using the [Approve Refund](/docs/manage-data-operations/lynx-api/refunds/approve-or-decline-a-refund) endpoint.
