## 检索机器的 ID

要通过序列号检索机器的 ID，您需要使用 [Get All Machines](/reference/lynx/machines/get-all-machine-basic-info) 端点。通过在请求中提供设备的序列号，您可以检索到相应机器的唯一标识符。

以下代码块展示了请求示例，使用 `DeviceSerial` 作为查询参数来过滤结果：

```sh
curl -X GET "https://qa-lynx.nayax.com/operational/v1/machines?DeviceSerial=<DEVICE_SERIAL_NUMBER>" \
-H "Authorization: Bearer <YOUR_ACCESS_TOKEN>"
```

<Note>
  Remember to replace `<DEVICE_SERIAL_NUMBER>` with the actual device's serial number.
</Note>

## Nayax Core

您也可以在 Nayax Core 中该机器的详细信息里找到 Machine ID，请按以下步骤操作：

1. 访问 Operations > Machines。
2. 使用搜索功能查找该机器。
3. 从列表中选择该机器。
4. 访问 General Information 选项卡，在 Hardware 部分查找 Machine ID。
