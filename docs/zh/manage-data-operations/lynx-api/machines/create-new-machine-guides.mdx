在 Nayax Core 中，您可以在**“机器”**菜单中创建新机器。请按照以下步骤操作。

1. 在您的 Dashboard 中前往 Operations > Machines。
2. 点击 Create > New Machine。

<Frame>
  <img src="/images/docs/create-new-machine-guides/image2.png" />
</Frame>

3. 现在您可以填写表单中的所有详细信息，完成后点击 Save。

Lynx API 允许您通过 POST 请求以编程方式完成此过程。本页面将指导您如何使用以下端点执行此操作：

<EndpointCard title="创建新机器" href="/reference/lynx/machines/create-new-machine" method="post" />

<Warning>
  验证

  请参阅本文档的[“安全和令牌”](/docs/manage-data-operations/lynx-api/security)页面，了解如何访问您的令牌，以及如何正确使用它来验证您的 API 请求。
</Warning>

您需要在正文参数中提供多个必填字段，例如：

| 字段名称               | 描述                                                                                                                                                                                                                                                                |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ActorID`          | 创建机器时所使用的操作员 ID。                                                                                                                                                                                                                                                  |
| `CountryID`        | 代表机器所在国家/地区的 ID。使用[“获取国家/地区”](/reference/lynx/lookups/get-countries)端点获取可用国家/地区代码列表。                                                                                                                                                                              |
| `CurrencyID`       | 机器货币的 ID。使用[“获取货币”](/reference/lynx/lookups/get-currencies)端点获取可用货币代码列表。                                                                                                                                                                                          |
| `MachineGroupID`   | 机器所属分组的 ID。您可以使用 [List All Machine Groups for Actor](/reference/lynx/actors/list-all-machine-groups-for-actor) 端点查看现有分组。                                                                                                                                          |
| `MachineModelID`   | ID for the machine's model. To obtain the machine model of an existing machine, you can copy and paste the `MachineModelId` from the [Get Specific Machine Basic Info](/reference/lynx/machines/get-specific-machine-basic-info) endpoint for a specific machine. |
| `MachineName`      | 虚拟机器的名称。                                                                                                                                                                                                                                                          |
| `MachineStatusBit` | 机器的状态：`1` 表示活跃，`2` 表示非活跃。                                                                                                                                                                                                                                         |
| `LanguageID`       | 该 ID 表示机器所使用的语言。                                                                                                                                                                                                                                                  |
| `SalesSourceID`    | 表示销售来源的 ID。应为 `30000512`，即表示实时交易（其他选项无关紧要）                                                                                                                                                                                                                        |
| `MachineTypeID`    | 机器的类型决定了其在 Nayax Core 中显示的图标。                                                                                                                                                                                                                                     |
| `GeoCountry`       | 机器所在国家/地区的代码。                                                                                                                                                                                                                                                     |

您将收到一个包含新创建机器详细信息的响应，其中包括新创建机器的 `MachineID`。

## 验证创建

新机器创建完成后，您可以在 Nayax Core 中进行验证，具体步骤如下：

1. 前往**“操作” > 机器**。
2. 使用 Search 功能筛选机器，并从列表中选择该机器。

<Frame>![](/images/docs/create-new-machine-guides/image1.png)</Frame>

如上图所示，选中后您可以在 General Information 选项卡中查看其所有信息。
