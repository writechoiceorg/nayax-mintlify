在 Nayax Core 中，您可以使用**“操作员”**菜单创建和管理操作员。要创建新的操作员，请按照以下步骤操作：

1. 前往**管理 > 操作员**。
2. 单击**“创建” > 添加新操作员**。
3. 现在，您必须填写表格上所需的详细信息。
   1. **操作员内部代码**：这是 SAP 客户编号。
   2. **父级**：指这个新运营商的父级参与者，在这个例子中，它就是分销商本身。
   3. **类型**：运算符类型。
   4. **状态**：操作员的状态。
   5. **区域设置**：有关货币和时区的配置。
4. 完成后，单击**“保存信息”** 。

<Warning>
  Actor 类型

  请注意，当您在分销商的正下方创建参与者时，只有两种类型可用：机构或运营商。要创建路线、区域或位置参与者类型，它们必须位于具有运营商类型的参与者下方。
</Warning>

### 创建操作员请求

[创建新的 Actor](/reference/lynx/actors/create-a-new-actor-1)端点允许在 Nayax 系统中注册新的操作员。请参阅以下代码块中的示例请求：

```sh Request
curl --request POST \
     --url https://qa-lynx.nayax.com/operational/v1/actors/{ParentActorID} \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json' \
     --header 'Content-Type: application/json' \
     --data '{
       "ActorDescription": "Main Branch",
       "ActorCode": 1001,
       "ActorAddress": "123 Main Street, Metropolis",
       "ActorContact": "contact@branch.com",
       "ActorTypeID": 16,
       "ActorStatus": 1,
       "CurrencyID": 840,
       "CountryID": 840,
       "ActorMerchantBit": true,
       "ActorCreationDate": "2024-11-25T12:00:00Z",
       "ActorLanguageID": 7,
       "GeoCity": "Metropolis",
       "GeoLongitude": -77.0364,
       "GeoLatitude": 38.8951
     }'

```

<Note>
  路径参数

  将 `ParentActorID` 更改为将在其下创建新运营商的父级 Actor 的唯一标识符。
</Note>

在body参数中填写以下内容：

| 范围            | `Type`           | 描述                       |
| ------------- | ---------------- | ------------------------ |
| ParentActorID | `int64/null`     | 如果该参与者是子实体，则为父参与者的唯一标识符。 |
| 演员ID          | `int64/null`     | 演员的唯一标识符。                |
| 演员描述          | `string/null`    | 演员的简要描述。                 |
| 演员代码          | `int64/null`     | 代表参与者的代码通常用于识别或分类。       |
| 演员地址          | `string/null`    | 与演员相关的物理地址。              |
| 演员联系方式        | `string/null`    | 演员的联系信息，例如电话号码或电子邮件。     |
| 演员类型ID        | `int32/null`     | 该标识符代表参与者的类型。            |
| 演员状态          | `int32/null`     | 演员的当前状态。                 |
| 货币ID          | `int32/null`     | 参与者用于交易的货币标识符。           |
| 国家/地区ID       | `int32/null`     | 演员所在国家/地区的标识符。           |
| 演员MerchantBit | `boolean/null`   | 指示参与者是否与商家关联。            |
| 演员语言ID        | `int32/null`     | 演员的语言偏好。                 |
| 地理城           | `string/null`    | 演员所在的城市。                 |
| 地理经度          | `double/null`    | 演员所在地的地理经度。              |
| 地理纬度          | `double/null`    | 演员所在地的地理纬度。              |
| 演员创作日期        | `date-time/null` | 创建此演员的日期和时间。             |

成功的请求将返回包含创建的操作员详细信息的响应。

## 验证创建

您可以在 Nayax Core 的 `Operators` 菜单中检查您的运营商是否已正确创建。

1. 使用**搜索**功能来过滤您的运营商。
2. 从列表中选择它。
