## 为卡片充值的方法

为卡片充值会增加其可用余额，使持卡人能够进行交易。Lynx API 提供了几种实现方式：

- PUT Method：使用此选项，您可以使用与创建或检索卡片数据相同的端点，但改用带有新充值金额的 PUT 请求。例如，您可以使用 [Update Card by ID](/reference/lynx/cards/update-card-by-id) 端点，在 URL 中提供 `CardID`，并在正文参数中提供更新后的充值金额。
- POST 方法：Lynx API 还提供了一个专门的端点，用于执行 POST 请求以向卡片添加信用额度。这是一个更简单的选项，因为您无需处理 PUT 方法所涉及的更大的 body 对象。

在本指南中，我们将介绍使用 POST 方法为卡片添加信用额度的方法。但您也可以查看 Updating Card Details 页面，了解有关 PUT 请求的更多详细信息。

## 添加信用额度

要为卡片添加信用额度，您可以在 [Add Credit to a Card](/reference/lynx/cards/add-credit-to-a-card) 端点上使用 POST 方法，并在 URL 中发送卡片的唯一标识符（Unique Identifier）。请参见下方的示例查询：

<CodeGroup>
  ```sh Request
  curl --request POST \
       --url 'https://qa-lynx.nayax.com/operational/v1/cards/{CardUniqueIdentifier}/credit/add?CardCredit=50&CreditChangeRemarks=Adding%20credit' \
       --header 'accept: application/json'
  ```

  ```json Response
  {
    "value": 50
  }
  ```
</CodeGroup>

<Note>
  将 `{CardUniqueIdentifier}` 替换为该卡的实际标识符。
</Note>

要添加的金额通过查询参数发送，请参见下方列表中的可用参数：

- `CardCredit`：要添加的信用额度金额。
- `CreditChangeRemarks`：用于追踪目的的交易备注。

## 在 Nayax Core 中验证信用额度

添加卡片信用额度的请求成功后，您可以前往 Nayax Core，在 Card Management 选项卡中查看该卡片的信用额度和历史记录。请按照以下步骤操作。

1. 访问您的 Nayax Core 仪表板

2. 在顶部菜单上，导航至**消费者 > 卡片管理**

   <Frame>
     <img src="/images/docs/add-credit-to-card/image1.png" />
   </Frame>

3. 搜索添加了信用额度的卡片并选择它。

   <Frame>
     <img src="/images/docs/add-credit-to-card/image2.png" />
   </Frame>

4. 打开 Change Credit 选项卡，并查看 Credit History 部分。

   <Frame>
     <img src="/images/docs/add-credit-to-card/image3.png" />
   </Frame>

在那里，您将能够看到该卡片的 Credit History（信用额度历史记录），找到通过 Lynx API 添加的信用额度，并将金额或备注与请求中发送的内容进行匹配。
