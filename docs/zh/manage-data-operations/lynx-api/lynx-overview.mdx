Lynx 是 Nayax 的管理 API。它允许您与 Nayax Core 集成，并在您自己的环境中管理机器、设备和库存。本文档的这一部分提供了指南和资源，帮助您使用 Lynx API 执行各种任务，例如库存管理、机器配置等等。

## 环境

本节所有指南示例中均使用测试环境 URL。部署到生产环境时，请替换主机名。

| 环境   | 基本 URL                                  |
| ---- | --------------------------------------- |
| 质量保证 | `https://qa-lynx.nayax.com/operational` |
| 生产   | `https://lynx.nayax.com/operational`    |

## Lynx API 的主要优势

Lynx API 为寻求自动化和简化运营的企业提供了广泛的优势：

- **简单集成**：轻松将 Lynx API 与您现有的内部系统集成。
- 全面的功能:执行各种操作,包括:
  - 库存管理和优化
  - 机器和设备配置
  - 执行远程操作和命令
  - 管理预付卡

## Lynx用例

Lynx API 支持多种实际用例，例如：

- **ERP系统集成**：与ERP系统集成，实现实时更新和管理。
- **预付卡充值**：使用自动化工作流程向预付卡中添加信用额度。
- **消费者管理**：自动化工作场所或零售环境中的预付卡管理。
- **库存管理**：创建拣货清单、执行库存更新并优化多台机器的库存水平。
