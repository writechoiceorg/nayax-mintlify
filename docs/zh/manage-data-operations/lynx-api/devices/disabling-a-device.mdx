有时，您可能希望禁用与机器关联的设备，以阻止其处理付款和订单。您可以使用以下端点，通过 Lynx API 以编程方式快速完成此操作。

## 禁用设备

使用 [Update Device](/reference/lynx/devices/update-device) 端点更新设备的状态。您需要提供 `DeviceID`、`ActorID`（运营商 ID）以及新的状态标志。请参见下方代码块中的示例请求：

```sh
curl -X PUT "https://qa-lynx.nayax.com/operational/v1/devices/<DEVICE_ID>" \
-H "Authorization: Bearer <YOUR_ACCESS_TOKEN>" \
-H "Content-Type: application/json" \
-d '{
  "ActorID": "<ACTOR_ID>",
  "StatusID": 2
}'
```

<Note>
  路径参数

  Replace `<DEVICE_ID>` with the actual identifier of the device you want to disable.
</Note>

在 Body 参数中，您需要提供以下内容：

| 身体参数       | 类型      | 描述                                         |
| :--------- | :------ | :----------------------------------------- |
| `ActorID`  | `int64` | 与该设备关联的运营商的标识符。                            |
| `StatusID` | `int32` | 设备的状态标识符：1 = Active（激活），2 = Inactive（未激活）。 |

因此，要禁用设备，请将 `StatusID` 更改为 2。响应将显示该设备的详细信息，确认设备状态已被禁用。

## 验证更新

您还可以使用 [Get Device by DeviceID](/reference/lynx/devices/get-device-by-deviceid) 端点检索设备信息，以验证更新是否成功。

另一种验证方法是在 Nayax Core 中，您可以查看每个设备的详细信息并验证其当前状态。请按照以下步骤操作：

1. 在您的仪表板中，转到**管理 > 设备**。
2. 使用搜索功能来筛选设备列表。
3. 从列表中选择所需的设备。
4. 打开**“常规信息”**选项卡并查看**“状态”**字段。
