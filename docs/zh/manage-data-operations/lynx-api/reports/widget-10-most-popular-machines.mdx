“10 Most Popular Machines Sales/Vends” 组件提供了您机队中表现最佳机器的排名。它使运营商能够根据销售额快速识别高客流量的位置和最盈利的设备。

<Frame>
  <img src="/images/docs/widgets-lynx/image7.jpg" />
</Frame>

以下是可通过此组件获取的信息：

1. Machine Name（Y 轴）：列出表现最佳机器的具体名称或 ID。
2. Sales Volume（X 轴）：以水平柱状图表示每台机器的相对业绩表现。
3. Time Period Label：显示当前选定的报表时间窗口（例如 Yesterday）。

<Accordion title="过滤器">
  - Chart Style：选择数据可视化格式，例如柱状图。
  - 类别轴主网格线：用于显示或隐藏垂直背景线以提高可读性的切换开关。
  - Include Cash：一个复选框，用于在机器的总计中加入或移除实体现金交易。
  - Time Period：定义排名所依据的日期范围，例如 Yesterday、Last 7 Days 或 Last 30 Days。
