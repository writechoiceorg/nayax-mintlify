---
title: 仪表盘销售表
description: 一个表格组件，提供标准报表时间窗口内营收和交易笔数的数值细分。
---

“Dashboard Sales Table” 提供按期间划分的详细业绩数值明细。它使运营商能够一目了然地查看特定时间段的确切营收和出货笔数。

<Frame>
  <img src="/images/docs/widgets-lynx/image3.jpg" />
</Frame>

以下是可通过此组件获取的信息：

1. Time Period：将数据细分为标准报表时间窗口：Today、This Week、Last Week、This Month、Last Month、This Year 和 Last Year。
2. Sales：显示按支付方式（例如 Credit Card）分类的营收金额以及该期间的总计。
3. Transactions：列出按支付方式分类的单次出货总笔数，以及该期间的总计。
