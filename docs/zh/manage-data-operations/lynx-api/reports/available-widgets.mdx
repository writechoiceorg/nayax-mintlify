---
title: 小部件 ID 列表
---

Widget Type ID 是每个组件的唯一数字标识符。它是所有 API 数据请求所需的主键，因为 `widget_type_id` 参数会明确指定您希望从应用程序代码中提取哪个组件的数据。

下表按范围列出了所有可用的组件 ID。请根据您的使用场景查找所需的 ID。
