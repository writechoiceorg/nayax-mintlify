## 获取拣货单

使用 [Get Pick List](/reference/lynx/machine-inventory/get-pick-list) 端点检索与特定机器关联的拣货单。

```sh Request
curl --request GET \
     --url https://qa-lynx.nayax.com/operational/v1/machines/{MachineID}/pickList \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json'
```

<Note>
  路径参数

  将 `MachineID` 更改为要从中检索拣货单的机器的实际 ID。
</Note>

成功的响应会返回该拣货单以及与机器关联的所有产品。

## 更新拣货单

使用 [Update Pick List](/reference/lynx/machine-inventory/update-pick-list) 端点修改机器拣货单中的产品数量或其他详细信息。

```sh Request
curl --request PUT \
     --url https://qa-lynx.nayax.com/operational/v1/machines/inventory/picklists/update \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json' \
     --header 'Content-Type: application/json' \
     --data '{
       "MachineId": 12345,
       "Products": [
         {
           "MachineProductId": 67890,
           "PickQty": 10
         },
         {
           "MachineProductId": 54321,
           "PickQty": 5
         }
       ]
     }'
```

请求正文包含以下参数：

| 范围          | 类型                 | 描述                                                                     |
| :---------- | :----------------- | :--------------------------------------------------------------------- |
| `MachineId` | `int64`            | 正在更新拣货单的机器的唯一标识符。                                                      |
| `Products`  | `array of objects` | 要更新的产品列表。每个对象包含 `MachineProductId`（产品在机器中的唯一标识符）和 `PickQty`（更新后的拣选数量）。 |

成功的响应会返回带有更新后拣选数量的拣货单。

## 删除拣货单

使用 [Delete Pick List](/reference/lynx/machine-inventory/delete-pick-list) 端点移除与特定机器关联的拣货单。

```sh Request
curl --request DELETE \
     --url https://qa-lynx.nayax.com/operational/v1/machines/{MachineID}/pickList \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json'
```

`200 OK` 响应确认拣货单已被删除。
