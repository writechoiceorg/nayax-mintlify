产品组将运营商的产品归类为诸如“Beverages”或“Snacks”之类的逻辑集群，使库存跟踪和补货更易于管理。

在 Nayax Core 中，您可以通过以下方式创建产品组：

1. 转到 Administration > Products。
2. 点击 Create > Add Product Group。
3. 在表单中填写所需的产品组详细信息：
   - 操作员：选择与该组关联的操作员。
   - Description：产品组的名称。
4. 点击“保存”以完成。

## 通过 API 创建产品组

使用 [Create New Product Groups](/reference/lynx/product-groups/create-new-product-groups) 端点以编程方式创建产品组。

<EndpointCard title="创建新产品组" href="/reference/lynx/product-groups/create-new-product-groups" method="post" />

<Warning>
  验证

  请参阅本文档的[“安全和令牌”](/docs/manage-data-operations/lynx-api/security)页面，了解如何访问您的令牌以及如何正确使用它们来验证您的 API 请求。
</Warning>

<br />

发送带有以下正文参数的 POST 请求：

```sh
curl --request POST \
     --url https://qa-lynx.nayax.com/operational/v1/productGroups \
     --header 'Authorization: Bearer <YOUR_ACCESS_TOKEN>' \
     --header 'accept: application/json' \
     --header 'Content-Type: application/json' \
     --data '{
       "ActorID": 12345,
       "ProductGroupName": "Beverages",
       "ProductGroupCode": 1001,
       "ProductGroupSubCode": 10,
       "ProductGroupCategoryCode": "DRINKS"
     }'
```

| 范围      | `Type`        | 描述                         |
| ------- | ------------- | -------------------------- |
| 演员ID    | `int64`       | 与产品组关联的参与者的唯一标识符。          |
| 产品组名称   | `string/null` | 要创建的产品组的名称（例如“Beverages”）。 |
| 产品组代码   | `int64/null`  | 与产品组关联的代码。                 |
| 产品组子代码  | `int64/null`  | 用于进一步对产品组进行分类的子代码。         |
| 产品组类别代码 | `string/null` | 与产品组关联的类别代码（例如“DRINKS”）。   |

After creating your product groups, you're ready to [create products](/docs/manage-data-operations/lynx-api/inventory-management/create-products).
