要设置 Amazon SQS 与 Nayax 的集成，我们建议您浏览以下文档页面：

1. [**Create New Queue**](/docs/manage-data-operations/amazon-sqs/amazon-queue-setup/create-new-queue): Learn how to set up a queue to receive data securely and efficiently.
2. [**Set Up Queue Permissions**](/docs/manage-data-operations/amazon-sqs/amazon-queue-setup/set-up-queue-permissions): This guide covers configuring access controls to manage queue security.
3. [**Create User for Queue Access**](/docs/manage-data-operations/amazon-sqs/amazon-queue-setup/create-user-for-queue-access): Follow this guide to create and configure an IAM user with the appropriate permissions.

## 先决条件

在继续之前，请确保以下事项：

- You have an [**Amazon AWS account**](https://aws.amazon.com/free/?all-free-tier.sort-by=item.additionalFields.SortRank\&all-free-tier.sort-order=asc).
- 您已登录到**AWS 管理控制台**。
