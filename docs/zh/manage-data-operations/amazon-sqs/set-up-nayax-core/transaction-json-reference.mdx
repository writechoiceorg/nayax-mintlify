每次处理交易时，Nayax Core 都会向您的 SQS 队列传递一条 JSON 消息。该消息包含两个层级：

- 根字段（`TransactionId`、`MachineId`、`SiteId` 等）在较高层级上标识该交易和机器。
- `Data` 对象包含完整的交易详情，包括支付方式、卡片信息、时间戳、定价，以及一个列出每个已售出商品的嵌套 `Products` 数组。

下方的 [JSON Message Fields Description](#json-message-fields-description) 表格中包含每个字段的完整说明。

## JSON 消息字段描述

| 字段名称                                           | 类型          | 描述                                                                                    |
| ---------------------------------------------- | ----------- | ------------------------------------------------------------------------------------- |
| 机器名称                                           | `String`    | 机器/POS 名称，如 Operations > Machines > Machine Name 中所定义                                 |
| 机器型号                                           | `String`    | 所用机器的型号。                                                                              |
| 运营商标识符                                         | `String`    | 机器/POS 编号，如 Operations > Machines > Machine Number 中所定义                               |
| Machine AuTime（机器授权时间）                         | `Date Time` | 基于机器所在时区的机器授权日期和时间                                                                    |
| Machine SeTime（机器结算时间）                         | `Date Time` | 基于机器所在时区的机器结算日期和时间                                                                    |
| Currency（货币）                                   | `String`    | 基于机器配置的交易货币                                                                           |
| Card String（卡片字符串）                             | `String`    | 支付卡的前 4 位和后 4 位数字                                                                     |
| Brand（品牌）                                      | `String`    | 信用卡品牌名称                                                                               |
| CLI                                            | `String`    | 消费者电话号码，适用于通过移动端进行的交易                                                                 |

| Payment Service（移动端使用信用卡支付）                    | `String`    | 所使用的支付服务，例如通过移动端使用信用卡支付。                                                              |
