Nayax Core 支持通过多种渠道交付交易和运营数据。您可以配置实时交付至 Amazon SQS，或设置定期批量交付至 SFTP 服务器或电子邮件地址。

要设置 Nayax Core，我们建议您查看以下文档页面：

- [**Transaction Report Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-transaction-delivery): Configure real-time delivery of transaction data to Amazon SQS, or set up periodic delivery via SFTP or Email.
- [**DEX Audit File Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-eva-dts-dexddcmp-audit-file-delivery): Enable audit file delivery in compliance with EVA DTS standards.
- [**Event and Alert Delivery**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-events-delivery): Configure delivery of system events for real-time tracking and monitoring.
- [**SQS Message Encryption**](/docs/manage-data-operations/amazon-sqs/set-up-nayax-core/set-up-encryption-on-amazon-sqs-delivery): Implement encryption for secure data transmission.

按照这些指南，您可以为所有支持的交付方式完全配置 Nayax Core。
