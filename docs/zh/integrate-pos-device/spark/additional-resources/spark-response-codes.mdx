以下是响应代码的完整列表，包括其描述和相关的 Spark API 方法。此表可帮助您了解 API 调用的结果，从成功的交易到各类错误类型。

| 响应代码  | 描述                                                    | 相关 API 方法                                                                                                                                                                                  |
| ----- | ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `20`  | 机器无设备（Machine Has No Device）                          | `/TriggerTransaction`                                                                                                                                                                      |
| `21`  | 设备未连接到 MQTT 服务器（Device Not Connected to MQTT Server）  | `/TriggerTransaction`                                                                                                                                                                      |
| `22`  | 缺少设备 IP（Missing Device IP）                            | `/TriggerTransaction`                                                                                                                                                                      |
| `23`  | IP 为非 VPN（IP Is Non-VPN）                              | `/TriggerTransaction`                                                                                                                                                                      |
| `24`  | 机器无响应（Machine Not Responding）                         | `/TriggerTransaction`                                                                                                                                                                      |
| `25`  | 未知机器 ID（Unknown Machine ID）                           | `/TriggerTransaction`                                                                                                                                                                      |
| `26`  | 验证错误（Validation Error）                                | `/StartAuthentication`                                                                                                                                                                     |
| `27`  | 未知交易 ID（Unknown Transaction ID）                       | `/Settlement`, `/CancelTransaction`                                                                                                                                                        |
| `28`  | 已被取消（Already Cancelled）                               | `/CancelTransaction`                                                                                                                                                                       |
| `29`  | 不允许唤醒设备（Not Allowed to Wake Device）                   | `/TriggerTransaction`                                                                                                                                                                      |
| `30`  | Spark 集成方未激活（Spark Integrator Not Active）             | `/StartAuthentication`                                                                                                                                                                     |
| `31`  | 取消失败（Cancellation Failed）                             | `/CancelTransaction`                                                                                                                                                                       |
| `32`  | 交易未找到或已过期（Transaction Not Found or Old）               | `/Settlement`, `/CancelTransaction`                                                                                                                                                        |
| `33`  | 身份验证失败（Fail To Authenticate）                          | `/StartAuthentication`                                                                                                                                                                     |
| `34`  | 缺少金额（Missing Amount）                                  | `/TriggerTransaction`、`/Settlement`                                                                                                                                                        |
| `35`  | 结算失败                                                  | `/Settlement`                                                                                                                                                                              |
| `36`  | 未找到站点 ID（Site ID Not Found）                           | `/Settlement`, `/CancelTransaction`                                                                                                                                                        |
| `37`  | 授权失败                                                  | [`/TransactionCallback`](/docs/integrate-pos-device/spark/webhook-callbacks/transactioncallback)                                                                                           |
| `38`  | 交易回调失败（Transaction Callback Failed）                   | [`/TransactionCallback`](/docs/integrate-pos-device/spark/webhook-callbacks/transactioncallback), [`/DeclineCallback`](/docs/integrate-pos-device/spark/webhook-callbacks/declinecallback) |
| `40`  | 超时回调失败（Timeout Callback Failed）                       | [`/TimeoutCallback`](/reference/spark/transactions/timeoutcallback)                                                                                                                        |
| `44`  | POS 设备超时（Timeout on POS Device）                       | [`/DeclineCallback`](/docs/integrate-pos-device/spark/webhook-callbacks/declinecallback)                                                                                                   |
| `45`  | 在 POS 设备上已取消（Cancelled on POS Device）                 | [`/DeclineCallback`](/docs/integrate-pos-device/spark/webhook-callbacks/declinecallback)                                                                                                   |
| `50`  | 延迟结算失败（Deferred Settlement Failed）                    | /ExternalSettlement                                                                                                                                                                        |
| `51`  | 延迟取消失败（Deferred Cancel Failed）                        | /ExternalCancel                                                                                                                                                                            |
| `52`  | 不支持延迟结算/取消（No Support for Deferred Settlement/Cancel） | /ExternalSettlement、/ExternalCancel                                                                                                                                                        |
| `992` | Nayax 超时错误（Nayax Time Out Error）                      | `All`                                                                                                                                                                                      |
| `997` | Nayax 验证错误（Nayax Validation Error）                    | `All`                                                                                                                                                                                      |
| `999` | Nayax 一般错误（Nayax General Error）                       | `All`                                                                                                                                                                                      |
