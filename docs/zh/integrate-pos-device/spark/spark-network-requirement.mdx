对于 Nayax 和集成器之间的通信，我们支持使用 TLS 1.2 和 1.3 的 HTTPS。

## 将 IP 地址加入白名单

为了确保通信顺畅，集成商应将以下 Nayax 公共 IP 列入白名单：

- 生产服务器：
  - DE（德国）：137.221.37.126、185.159.234.2
  - 美国（美国）：72.28.118.100、72.28.111.132、185.159.233.2
  - 伊利诺伊州（以色列）：185.159.232.2、84.110.125.194、82.102.172.206、212.179.76.198
- QA 服务器：31.154.55.2

如果需要，集成商还应允许 Nayax QA 团队从 Nayax QA 环境访问以进行自动化测试。
