本节提供 Spark 集成中所有可用 API 方法的详细参考文档。每个方法均包含其用途、请求结构、响应结构和关键字段。在构建或维护您与 Spark 的集成时，请使用此参考文档。

所有请求都应遵循 [Security & Authentication](/docs/integrate-pos-device/spark/security-authentication/spark-authentication) 页面中所述的安全和身份验证准则。大多数端点都需要经过签名的请求头和有效的 Spark 交易标识符。
