import { SparkAuthTool } from "/snippets/SparkAuthTool.jsx";

使用此工具无需编写任何代码即可处理 Spark 身份验证。一切都在您的浏览器中运行，因此您的令牌和签名密钥不会离开您的机器。

顶部的选项卡涵盖了该工具执行的三种操作：

- **StartAuthentication**: build or inspect the cipher and the
  `TransactionSignature` header for the `StartAuthentication` call, the
  integrator's first request to open a transaction. The **Auth** and **Legacy
  Auth (v1)** sub-tabs cover the current and legacy signature flows.
- **StartSession**: build or inspect the `Cipher` your server returns when
  Nayax sends a `StartSession` request to open a session.
- Header Signature：生成用于验证 Spark API 请求身份的 `IntegratorId` 和 `TransactionSignature` 请求头。
