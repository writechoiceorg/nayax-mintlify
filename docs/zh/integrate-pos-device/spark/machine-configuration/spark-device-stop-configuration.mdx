title: 设备停止配置
description: 在 Nayax Core 中配置一台机器，以便在 Remote Start 或 Device Start 功能旁支持 Spark Device Stop。
---

本页面上的设置用于为 Spark Device Stop 准备机器，此流程下，终端处出示的卡片可停止一笔进行中的交易。Device Stop 与同一机器上的 Remote Start 或 Device Start 并行运行，因此请在您所使用的启动流程配置基础上应用这些设置。

## 读卡器

Transaction Start Method 设置定义了读卡器在空闲状态下可通过什么方式启动交易。将其设置为 Accept All，以便读卡器在空闲模式下接受卡片，从而触发 Device Stop 流程。

<Note>
  当您将 Device Stop 与 Remote Start 配对使用时，此属性必须为 Accept All，而不能是 Accept Nothing：当两者一起运行时，由 Device Stop 决定该值。
</Note>

## VPOS Touch

VAS configurations 设置用于告知服务器该机器支持 Spark Device Stop 功能。请根据与 Device Stop 配对的启动流程进行设置：

- Remote Start with Device Stop：设置为 Spark Device Stop，或同时设置 Spark Device Stop 和 Spark Remote Start。两种方式均可。
- Device Start with Device Stop：仅设置为 Spark Device Start (V2)。Device Start (V2) 值同时涵盖 Device Start 和 Device Stop。

## 配置摘要

根据与 Device Stop 配对的启动流程，使用此表作为快速参考：

| 团体         | 属性     | Remote Start with Device Stop                               | Device Start with Device Stop                                          |
| :--------- | :----- | :---------------------------------------------------------- | :--------------------------------------------------------------------- |
| 读卡器        | 交易启动方式 | Accept All                                                  | 单站点：Button+Contactless+Contact。多站点：Product                             |
| VPOS Touch | VAS 配置 | Spark Device Stop（或 Spark Device Stop 和 Spark Remote Start） | Spark Device Start (V2)（或 Spark Device Stop 和 Spark Device Start (V2)） |
