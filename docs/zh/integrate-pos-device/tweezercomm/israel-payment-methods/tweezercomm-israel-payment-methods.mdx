本节介绍使用 `ashrait` 支付服务在以色列处理交易的 API 方法，该服务允许您执行以下操作。

- 开始不同类型的交易。
- 取消正在进行但尚未完成的交易。
- 使用唯一 ID 查找过去交易的详细信息。
- 获取各种交易记录的报告。

这些工具可让您全面掌控支付流程，轻松管理、追踪并报告所有交易。
