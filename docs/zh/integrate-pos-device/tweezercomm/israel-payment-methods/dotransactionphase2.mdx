`doTransactionPhase2` 方法用于完成由 `doTransactionPhase1` 开始的交易。完成该交易仅需要 `vuid`。客户端可以更改在 `doTransactionPhase1` 调用中发送的任何项目。

## 要求

<Note>
  JSON-RPC 2.0

  TweezerComm 中的所有方法都遵循 JSON-RPC 2.0 结构。有关请求的结构和发送方式的更多详细信息，请参阅[“集成”部分](/docs/integrate-pos-device/tweezercomm/tweezercomm-get-started)。
</Note>

请在请求中确保配置以下内容：

- `method` 字段必须设置为 `doTransactionPhase2`。
- The service must be set to `ashrait` in the `params`.

该请求包含一个参数列表，其中包含所有特定于交易的详细信息。

```json
{  
   "jsonrpc":"2.0",
   "method":"DoTransactionPhase2",
   "params":[  
      "ashrait",
      {  
         "vuid":"123456789",
         "loyaltyAmount":5000,
         "creditTerms":1
      }
   ],
   "id":1
}
```
