本节介绍管理支付终端的 API 方法。您可以：

- 获取设备信息（如型号、序列号和软件版本）
- 读取卡数据（来自磁条或 NFC Mifare 卡）
- 配置终端设置（例如打印机使用或交易类型）
- 发送诊断日志以进行故障排除

这些工具可帮助您集成和控制支付设备，用于各种用途，包括会员计划、身份识别和系统维护。您还可以根据需要停止读卡操作。
