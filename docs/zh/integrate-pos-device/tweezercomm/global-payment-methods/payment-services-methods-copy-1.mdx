本节介绍使用 `engine` 支付服务在全球范围内处理支付交易的 API 方法。您可以：

- 开始不同类型的交易。
- 取消正在进行但尚未完成的交易。
- 使用唯一 ID 查找过去交易的详细信息。
- 获取各种交易记录的报告。

这些工具让您完全控制您的支付流程，让您轻松管理、跟踪和报告您的所有交易。
