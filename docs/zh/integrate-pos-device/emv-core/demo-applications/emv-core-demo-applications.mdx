本文提供了一个集中式的起点，用于测试 Nayax EMV Core 演示应用程序。Nayax 提供了一个演示应用程序，可帮助您从集成的第一天起验证硬件通信和交易流程，确保您的环境在开始自定义开发之前已正确配置。

### 为什么从演示应用开始？

- **立即验证：**确认您的读卡器和主机系统通信正常，无需编写新代码。
- **参考代码：**使用示例源代码作为蓝图来实现您自己的支付逻辑。
- **网关测试：**验证与 Nayax 网关的连接，并在受控环境中处理测试交易。
