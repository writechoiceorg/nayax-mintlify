Nayax OEM POS 解决方案提供预认证的 EMV 模块，可简化刷卡自助终端的开发。通过集成 EMV 核心模块，您可以专注于应用程序的 UI 和业务逻辑，而我们将负责处理复杂的交易处理。

<Note>
  版本兼容性

  本文档适用于 EMV Core v1.4，即当前版本。
</Note>

## 识别您的设备

安装完成后，您必须[获取设备 ID](/docs/integrate-pos-device/emv-core/get-started/emv-core-device-id) 。此唯一标识符用于与 Nayax 网关通信、初始化 EMV 核心以及开始处理测试交易。
