Marshall SDK 为使用 Nayax 的 VPOS Touch 和 VPOS Media 设备提供了简单全面的解决方案，免去了您处理 Marshall 协议细节的繁琐步骤。只需按照快速入门演示操作，即可快速上手。

<Danger>
  重要提示

  Nayax 仅支持使用我们的 SDK 完成的集成，并且这些集成已经过认证流程（更多信息请参见“认证流程”页面）。
</Danger>

所有 SDK 均提供全面的 API、详细的配置选项，并支持处理各种售货场景。以下指南将帮助您快速入门，并在项目中集成和配置这些 SDK，确保 Nayax 支付系统的顺利设置和运行。

<Note>
  可用语言

  为了您的方便，SDK 以 C、C# 和 JAVA 提供。
</Note>
