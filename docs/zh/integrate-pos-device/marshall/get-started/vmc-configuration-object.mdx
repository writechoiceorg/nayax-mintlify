`vmc_configuration` 类用于设置控制外围设备行为的各种参数和标志。正确的配置对于确保顺畅运行以及与不同外围设备的兼容性至关重要。

## Configuration Object

以下代码块显示了各可用 SDK 语言的 `vmc_configuration` 类。

## 必填字段

配置 VMC(自动售货机控制器)时,有一些特定的必填字段对于正确识别机器至关重要。妥善管理这些字段有助于改善故障排查、增强系统集成并提高维护效率。

请参阅下方代码块,查看这些必填字段的配置示例。

<CodeGroup>
  ```java Java
  vmc_config = new vmc_configuration();

  vmc_config.model = "marshall-java-sdk";
  vmc_config.serial = "01234567";
  vmc_config.hw_ver = "01234567";
  vmc_config.manuf_code = "manuf";
  ```

  ```csharp C#
  vmc_config = new vmc_configuration();

  vmc_config.model = "marshall-java-sdk";
  vmc_config.serial = "01234567";
  vmc_config.hw_ver = "01234567";
  vmc_config.manuf_code = "manuf";
  ```

  ```c C
  vmc_init()
  	__strcpy(config.model, "marshall-c-sdk-demo");
  	__strcpy(config.serial, "01234567");
  	__strcpy(config.sw_ver, "vmc app version");
  	__strcpy(config.vmc_hw_ver, “01234567”);
    __strcpy(config. vmc_manuf_code, “manuf”);
  ```
</CodeGroup>

下表描述了这些参数:

| 范围           | 类型     | 描述                                              |
| :----------- | :----- | :---------------------------------------------- |
| `model`      | String | 机器的型号名称。有助于在日志和报告中区分不同的机器类型,便于故障排查。             |
| `serial`     | String | 机器的序列号。这可实现精确追踪、有效的库存管理和简化的维护计划安排。              |
| `hw_ver`     | String | 机器的硬件版本。它确保使用正确的固件和软件。不同的硬件版本可能需要特定的驱动程序或兼容性调整。 |
| `manuf_code` | String | 标识机器的制造商,以便提供适当的支持和运行保障。                        |
