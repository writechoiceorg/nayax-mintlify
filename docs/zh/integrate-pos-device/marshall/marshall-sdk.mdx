Marshall 协议是 Nayax 公司开发的专有通信协议，基于 RS232 接口运行。该协议设计稳健可靠，支持机器控制器与 VPOST、ONYX 和 VPOSM 产品线之间进行可靠的交易交换。它提供功能丰富的支付 API，支持多种支付流程，例如：

- 预选
- 预授权
- 多会话
- 多机售卖

Marshall SDK 提供了一种简单而全面的方式来利用这些流程，无需管理 Marshall 协议的细节。

<McpNote />

<Note>
  需要加油机或加油站集成方案?

  本页介绍 Marshall，即 Nayax 用于自动售货机控制器 (VMC) 集成的 RS232 协议。有关加油机和油站控制器 (FCC) 集成，请参阅 [Marshall Pro](/docs/integrate-pos-device/marshall-pro/getting-started)。
</Note>

<Note>

  为了您的方便，SDK 以 C、C# 和 JAVA 提供。
</Note>

<br />

<Warning>
  价格字段/小数位数

  Marshall 中的价格字段是一个 2 字节的整数，这意味着可以发送到设备的最高值为 65535 (0xFFFF)。设备实际向消费者收取的费用是外围设备发送的价格乘以 10 的负小数位数：也就是说，设备在进行授权和结算的价格计算时，会参考其在 Nayax Core 中设置的自身小数位数配置（该值取决于设备安装所在的国家/地区）。

  这意味着，如果小数位数为 2，则该设备能够授权或向消费者收取的最高价格为 655.35。
</Warning>
