Nayax Marshall SDK 包含一个模拟器，可帮助开发者测试和调试其设备与 Nayax 支付系统的集成。该模拟器能够模拟实际硬件和软件交互的行为，使您无需使用物理硬件即可验证售货流程。

## 模拟器

在终端上启动 C# SDK 模拟器后,其菜单将如下所示:

<TerminalBlock>
  \[0]: 开始会话 0
  \[1]: 关闭会话 0
  \[2]: 会话信息
  \[3]: 取消会话
  \[4]: 查询会话
  \[5]: 读卡器状态
  \[6]: 现金销售
  \[7]: 警报
  \[8]: 传输自定义 transaction\_id
  \[9]: 品牌配置
  \[10]: FTP 套接字测试
  \[11]: 屏幕控制
  \[12]: 链路统计
  \[13]: 退出
</TerminalBlock>

## 交易流程示例

以下是每种交易类型的流程示例：

1. 常规交易（单一会话）：
   1. 启动会话：使用`[0]: Start session`。
   2. 出示银行卡，等待授权和“谢谢”屏幕。
2. 多会话交易：
   1. 使用`[0]: Start session 0`启动会话。
   2. 出示银行卡，等待授权和“谢谢”屏幕。
   3. 关闭相关会话——在此情况下，使用`[1]: Close Session 0`。
