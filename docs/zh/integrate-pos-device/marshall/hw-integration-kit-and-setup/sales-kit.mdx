VPOS Touch

以下组件是 Touch 设备安装的一部分。每个组件在安装和操作过程中都发挥着特定作用：

| 姓名                   | 描述                                      | 图像                                                    |
| :------------------- | :-------------------------------------- | :---------------------------------------------------- |
| Touch 设备             | Nayax VPOS Touch Marshall 设备。           | <Frame>![](/images/docs/sales-kit/image1.png)</Frame> |
| 设备连接器                | 连接电源、电缆以及设备运行所需的其他配件。                   | <Frame>![](/images/docs/sales-kit/image2.png)</Frame> |
| LAN(以太网电缆端口)         | 支持通过有线连接到路由器以获得互联网访问。                   | <Frame>![](/images/docs/sales-kit/image3.png)</Frame> |
| 天线                   | 连接蜂窝天线以进行网络通信。                          | <Frame>![](/images/docs/sales-kit/image4.png)</Frame> |
| SIM                  | 用于在 Nayax Touch VPOS 设备中启用数据连接的 SIM 卡。  | <Frame>![](/images/docs/sales-kit/image5.png)</Frame> |
| Chipi-X USB2Com 适配电缆 | 用于连接 Nayax Touch VPOS 设备的 USB 转串口适配电缆。  | <Frame>![](/images/docs/sales-kit/image6.png)</Frame> |
| 24V 直流电源适配器          | 用于为 Nayax Touch VPOS 设备供电的 24V 直流电源适配器。 | <Frame>![](/images/docs/sales-kit/image7.png)</Frame> |

VPOS Media4S

以下组件是 Media 设备安装的一部分。每个组件在安装和操作过程中都发挥特定作用:

| 姓名                       | 描述                                                                       | 图像                                                      |
| :----------------------- | :----------------------------------------------------------------------- | :------------------------------------------------------ |
| Media 设备                 | Nayax VPOS Media Marshall 设备。                                            | <Frame>![](/images/docs/sales-kit/image13.png)</Frame>  |
| 设备连接器                    | 连接电源、电缆以及设备运行所需的其他配件。                                                    | <Frame>![](/images/docs/sales-kit/image14.jpeg)</Frame> |
| Chipi-X USB2Com 适配电缆     | 用于连接 Nayax Touch VPOS 设备的 USB 转串口适配电缆。                                   | <Frame>![](/images/docs/sales-kit/image15.png)</Frame>  |
| 以太网适配器（可选）- USB-C 转 RJ45 | 可选。如果您希望设备通过以太网（ETH）经由 Marshall 与您的外围设备通信（而不是通过 Wi-Fi 或电缆进行串口连接），则此选项适用。 | <Frame>![](/images/docs/sales-kit/image16.jpg)</Frame>  |

<Note>
  **注意：** VPOSM4S 没有实体 SIM 卡，而是 eSIM，并且它还内置了天线。
</Note>
