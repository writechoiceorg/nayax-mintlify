## 推荐流程

如果您不确定哪种流量最适合您的机器类型，下表显示了推荐的流量和配置。

| 机器类型      | 推荐流量                                                                                                                                                             | 推荐配置                                       |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| 售货机       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |
| 硬币兑换器     | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |
| 棉花糖       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲 + 明确售卖成功                              |
| 照相亭       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |
| 咖啡机       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |
| 按摩椅       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |
| 洗车机       | [单会话 + 预选](/docs/integrate-pos-device/marshall/payment-flows/marshall-single-session-with-pre-sel)                                                               | 始终空闲                                       |

| 电动汽车充电器 2 | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)                                                                                  | 价格未最终确定 + Mifare 已获 VMC 批准                 |
| 电动汽车充电器 3 | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)                                                                                  | 价格尚未最终确定 + Mifare 已获 VMC 批准 + 信息发布会        |
| 电动汽车充电器 4 | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)                                                                                  | 价格尚未最终确定 + Mifare 已获 VMC 批准 + 信息发布会 + 始终开启 |
| 智能冰箱      | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)+[子流程/功能](/docs/integrate-pos-device/marshall/payment-flows/sub-flows-features#/) | 价格未最终确定 + 始终开启                             |
| 充电宝       | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)+[子流程/功能](/docs/integrate-pos-device/marshall/payment-flows/sub-flows-features#/) | 价格未最终确定 + 始终开启                             |
| 储物柜       | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)                                                                                  | 价格尚未最终确定                                   |
| 球拍租赁机     | [多会话](/docs/integrate-pos-device/marshall/payment-flows/marshall-multi-session)                                                                                  | 价格未最终确定 + 始终处于空闲状态                         |
