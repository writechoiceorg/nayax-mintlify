本页概述了 Marshall SDK 提供的通用功能。其中包括通过 `GeneralEvents` 接口进行事件处理，以及用于注册回调、发送警报和请求时间更新的 API。

## 事件接口

`GeneralEvents` 接口定义了与 Marshall SDK 一般操作相关的各种事件。实现此接口可使您的应用程序响应 SDK 运行期间触发的特定事件。以下代码块展示了所有可用的事件方法：

<CodeGroup>
  ```java Java
  public interface GeneralEvents
      void onDCSParamChange(int dcs_param, byte[] data);
      void onDisplayEventButtonPressed(byte button_id);
      void onDisplayEventTextInput(string input);
      void onTransferData(byte[] data);
      void onTimeUpdate(
              byte year,
              byte month,
              byte day,
              byte hours,
              byte minutes,
              byte seconds);
  ```

  ```csharp C#
  public interface GeneralEvents
      void onDCSParamChange(int dcs_param, byte[] data);
      void onTransferData(byte[] data);
      void onTimeUpdate(
              byte year,
              byte month,
              byte day,
              byte hours,
              byte minutes,
              byte seconds);
  ```
</CodeGroup>

## 一般方法

以下方法可用于与 Marshall SDK 的常规功能进行交互：

| 范围                                         | 类型   | 描述                                                  |
| :----------------------------------------- | :--- | :-------------------------------------------------- |
| `register_callbacks(GeneralEvents events)` | void | 为外设通用模块注册回调事件。此方法允许您的应用程序处理 GeneralEvents 接口中定义的事件。 |
| `alert(short id, String msg)`              | void | 向 Nayax Core 发送警报，显示在**“最新警报”**下。     |
| `requestTime()`                            | void | 向 VPOS 发起时间请求。响应将以事件形式（`onTimeUpdate`）到达。           |
