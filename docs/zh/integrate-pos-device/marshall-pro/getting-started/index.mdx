Marshall Pro（Fuel）协议是一个基于命令的 JSON 接口，旨在实现销售点（POS）或前台控制器（FCC）与 VPOS 支付终端之间的清晰通信。

该协议专为加油和服务行业而设计，使您的系统能够精准地管理整个客户旅程。它提供了一种统一的方式，通过稳定的网络连接，使用标准的 UTF-8 JSON 消息来处理硬件交互。

### 为什么选择 Marshall Pro？

该协议将复杂的加油站逻辑简化为可预测的 JSON 数据交换。无论您管理的是大型加油站还是独立加油亭，Marshall Pro 都能确保您的支付和会员流程保持同步和安全。
