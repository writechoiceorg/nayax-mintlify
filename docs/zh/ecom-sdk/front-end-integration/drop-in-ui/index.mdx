Drop-in UI 组件使您能够在网站的任何位置轻松显示一份完整、预配置的可用支付方式列表，从而最大限度地缩短集成开发时间。

该 UI 的其他功能包括：

- 新增的支付方式无需开发即可立即生效。
- 对支付方式列表拥有完整的样式控制权。
- 内置原生 3D Secure 2 支持。

<Note>
  实现 Drop-in UI

  `startPayment` 方法会自动处理 Drop-in UI 的实现。
</Note>

## 工作原理

Drop-in 的使用流程如下：

1. 消费者进入结账页面。
2. 这会使用来自服务器的会话数据创建一个 Drop-in 实例。
3. Drop-in 会显示可用的支付方式并收集消费者的支付详情。

   <Frame>
     <img src="/images/docs/drop-in-ui/image1.png" />
   </Frame>
4. 支付完成后，UI 会向消费者展示支付结果。

   <Frame>
     <img src="/images/docs/drop-in-ui/image2.png" />
   </Frame>

<Note>
  Webhook 通知

  Your webhook server simultaneously receives the event containing the payment outcome. See [Webhook Notifications](/docs/ecom-sdk/back-end-integration/webhook-notifications) for more details.
</Note>
