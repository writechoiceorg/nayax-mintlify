Nayax eCom SDK 是一款旨在简化商户电子商务支付处理的工具。它充当中介，简化您的前端应用程序与各种计费提供商之间的交互，使您能够：

- 接受全球及本地支付方式。

- 支持 Web、iOS 和 Android 的在线支付。

- 提供支付链接。

该 SDK 屏蔽了直接与多个支付网关集成的复杂性。商户无需为每个支付提供商构建自定义集成，只需与 Nayax eCom SDK 集成一次即可。

<McpNote />

## 支付方式

eCom SDK 支持以下支付方式：

- 签证
- 发现
- 万事达
- Diners
- Maestro
- Apple Pay
- Google Pay

## 集成架构

与 eCom 完整集成在线支付包含以下几个部分：

1. 支付服务器
   ：向 eCom API 端点发出 API 请求并接收响应。
2. 客户端网站或应用程序
   ：我们的 [Drop-in UI](docs:drop-in-ui) 从消费者处收集支付详细信息，并将支付信息传递给您的服务器。
3. **Webhook server**
   : Receives [webhooks](/docs/ecom-sdk/back-end-integration/webhook-notifications) from eCom about payment statuses and other payment-related activity.
