## Recommended Flows

If you are unsure which flow is ideal for you, in the table below you can see the recommended flow depending on your machine type, along with the recommended configuration for it.

| Machine Type          | Recommended Flow                                                           | Recommended Configuration                                           |
| --------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Vending Machine       | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| Coin Exchanger        | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| Cotton Candy          | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle + Explicit Vend Success                                 |
| Photo Booth           | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| Coffee Machine        | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| Massage Chair         | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| Car Washing Machine   | [Single Session + Pre-Selection](doc:marshall-single-session-with-pre-sel) | Always Idle                                                         |
| EV Charger 1          | [Multi Session](doc:multi-session-1)                                       | Price not final                                                     |
| EV Charger 2          | [Multi Session](doc:multi-session-1)                                       | Price not final + Mifare approved by VMC                            |
| EV Charger 3          | [Multi Session](doc:multi-session-1)                                       | Price not final + Mifare approved by VMC + Info Session             |
| EV Charger 4          | [Multi Session](doc:multi-session-1)                                       | Price not final + Mifare approved by VMC + Info Session + Always On |
| Smart Fridge          | [Multi Session](doc:multi-session-1)                                       | Price not final + Always On                                         |
| Power Bank            | [Multi Session](doc:multi-session-1)                                       | Price not final + Always On                                         |
| Locker                | [Multi Session](doc:multi-session-1)                                       | Price not final                                                     |
| Racket Rental Machine | [Multi Session](doc:multi-session-1)                                       | Price not final + Always Idle                                       |
