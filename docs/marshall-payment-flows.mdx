## Recommended Flows

If you are unsure which flow is ideal for your machine type, the following table shows the recommended flow and configuration.

| Machine Type          | Recommended Flow                                                                                                              | Recommended Configuration                                           |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Vending Machine       | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| Coin Exchanger        | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| Cotton Candy          | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle + Explicit Vend Success                                 |
| Photo Booth           | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| Coffee Machine        | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| Massage Chair         | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| Car Washing Machine   | [Single Session + Pre-Selection](/docs/marshall-single-session-with-pre-sel)                                                    | Always Idle                                                         |
| EV Charger 1          | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final                                                     |
| EV Charger 2          | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final + Mifare approved by VMC                            |
| EV Charger 3          | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final + Mifare approved by VMC + Info Session             |
| EV Charger 4          | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final + Mifare approved by VMC + Info Session + Always On |
| Smart Fridge          | [Multi Session](/docs/marshall-multi-session)+ [Sub flows / features](/docs/sub-flows-features#/) | Price not final + Always On                                         |
| Power Bank            | [Multi Session](/docs/marshall-multi-session)+ [Sub flows / features](/docs/sub-flows-features#/) | Price not final + Always On                                         |
| Locker                | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final                                                     |
| Racket Rental Machine | [Multi Session](/docs/marshall-multi-session)                                                                                   | Price not final + Always Idle                                       |
