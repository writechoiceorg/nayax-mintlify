---
title: "Actor Types"
---

The `ActorTypeID` parameter specifies the type of sub-operator creation in the Nayax Lynx system. The following table lists the possible values:

| Value | Actor Type          |
| ----- | ------------------- |
| 5     | Area                |
| 6     | Route               |
| 7     | City                |
| 8     | State               |
| 9     | Province            |
| 10    | Region              |
| 11    | Client              |
| 12    | Facility Management |
| 13    | Location (General)  |
| 14    | Customer (General)  |
| 15    | District            |
| 16    | Account             |
| 17    | Branch              |
| 18    | Terminal            |
| 19    | Floor               |
| 20    | Zone                |
| 21    | Airport             |
| 22    | Gate                |
| 23    | eCommerce           |
| 24    | Bus                 |
| 25    | Train               |
| 26    | Tramway             |
| 27    | EV Infrastructure   |
